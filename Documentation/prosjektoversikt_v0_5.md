# Prosjektoversikt v0.5

## Den korte forskningslinjen
- v0.1–v0.2: minimal relasjonell universgraf og token-basert dynamikk
- v0.3: utvidet feature-rom og skille mellom identiteter, invariants og quasi-invariants
- v0.4: redusert basis og regelbetingede \(\Delta F\)-matriser
- v0.5: perturbasjonslab, metodisk opprydding i lokalitet, og første operative causal-cone-analyse

## Hva som er robust etablert
1. Modellen kan formuleres som en stokastisk lokal grafdynamikk.
2. Enkelte regelklasser har eksakte lineære invariants.
3. En strengt lokal koplingsmetode for to replikaer er nå implementert.
4. Lokale forskjeller ser ut til å spre seg med begrenset rekkevidde i nær-konservative sektorer.

## Hva som fortsatt er åpent
1. Universell effektiv signalhastighet
2. Birth/death med kontrollert kopling
3. Sammenheng mellom causal cone og emergent dimensjon
4. Sammenheng mellom quasi-invarianter og fysisk energi

## Hva dette innebærer
Prosjektet er ikke lenger bare et konseptuelt rammeverk.
Det er nå et program for matematisk og simulativ testing av:
- lokalitet
- invariantsektorer
- og emergent kinematikk.
