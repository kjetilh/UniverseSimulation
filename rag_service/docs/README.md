# Dokumentasjonsoversikt

Denne mappen er den prosjektspesifikke dokumentpakken for `UniverseSimulation`-RAG-en.

Dokumentene her er skrevet for to formaal:

- mennesker som vil forsta hvor prosjektet star og hvordan verktøyene brukes
- sprakmodeller og research-klienter som trenger et ryddig, kildenaert korpus

## Viktigste dokumenter

- `UNIVERSE_RAG_STATUS.md`: hvor prosjektet faktisk er na
- `UNIVERSE_ARGUMENTATION_MAP.md`: prosjektets argumentasjonskjede og antagelser
- `UNIVERSE_TOOL_RUNBOOK.md`: hvordan dagens simulator og RAG brukes i praksis
- `UNIVERSE_CORPUS_PLAN.md`: anbefalt source_type- og case-struktur
- `UNIVERSE_DEEP_RESEARCH_PROMPT.md`: copy/paste-prompt for eksterne research-klienter
- `RAG_SERVICE_API.md`: praktisk oversikt over relevante endepunkter i denne tjenesten

## Leserekkefolge

Hvis du er ny i prosjektet:

1. les `UNIVERSE_RAG_STATUS.md`
2. les `UNIVERSE_ARGUMENTATION_MAP.md`
3. les `UNIVERSE_TOOL_RUNBOOK.md`
4. bruk `UNIVERSE_CORPUS_PLAN.md` nar du skal laste inn eller drifte korpuset

## Viktig arbeidsregel

Disse dokumentene er skrevet for a holde fire ting atskilt:

- hva forskningsrapporten sier
- hva koden faktisk gjor
- hva eksisterende kjoredata viser
- hva som bare er inferens eller neste hypotese
